# Twitter Crawler
트위터 트윗 크롤러 입니다.    
수집을 원하는 키워드와 수집할 기간을 설정하면 됩니다.


## 구현된 기능
1. 특정 키워드와 관련된 트윗 수집
2. 트윗 수집 기간 설정
3. 수집된 트윗 상세페이지 스크린샷 기능 


## 실행방법
```
$ yarn install
$ yarn dev
```
